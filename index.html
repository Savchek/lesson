Колір тексту
Aa
Користуватися вбудованими HTML-тегами звичайно зручно і легко, але часто є потреба більш тонко налаштувати характеристики відображення тексту.
Зазвичай це робиться за допомогою CSS стилів.
Для того щоб змінити колір тексту потрібно вказати властивіть color у відповідному селекторі.
Виглядати це буде приблизно так: color: purple;
Властивість color може приймати значення у декількох варіантах:
  1. Назва кольору. Наприклад blue, navy або lightblue;
  2. HEX значення кольору. Наприклад #1a24ad;
  3. RGB значення кольору. Наприклад rgb(26, 36, 173);
  4. HSL значення кольору. Наприклад hsl(145, 94%, 87%);

На прикладі нижче можна побачити декілька варіантів властивості кольору і яким чином вони впливають на текст.
Тут можна спробувати написати власну CSS властивість і подивитися як вона вплине на текст.
Не забувайте правила написання CSS властивостей! Двокрапка після властивості, крапка з комою після значення.
Приклад правильно написаної CSS властивості: color: rgb(146, 29, 214);
#color-text {
}
Вирівнювання тексту
Aa
Властивістю text-align можна задавати горизонтальне вирівнювання тексту. Властивість приймає наступні значення:
  1. left - вирівнювання по лівому краю (стандартне значення);
  2. right - вирівнювання по правому краю;
  3. center - вирівнювання по центру;
  4. justify - текст рівномірно розтягується по всій доступній йому ширині;

Декорація тексту
Aa
Властивістю text-decoration можна задавати декорацію тексту у вигляді стрічки. В залежності від значення стрічка буде в різних позиціях. Властивість приймає наступні значення:
  1. overline - стрічка над текстом;
  2. line-through - стрічка проходить скрізь текст;
  3. underline - стрічка під текстом;
  4. none - прибирає декорацию тексту;

Трансформація тексту
Aa

  3. capitalize - зводить перші літери слів до верхнього регістру, всі інші літери слів - до нижнього;

Інтервал тексту
Aбвгджз
До групи що впливають на інтервали у тексті відносяться декілька властивостей. Властивість letter-spacing відповідає за відстань між литерами.
Властивість word-spacing відповідає за відстань між словами.
Властивістю text-indent можна задати відступ першого речення у параграфі.
Властивість line-height задає висоту стрічки тексту не впливаючи на самий текст, у зв'язку з чим візуально змінюється відстань між рядками.
Всі ці властивості приймають як значення відстань у пікселях або інших одиницях виміровання CSS.

Також існує властивість - white-space - яка встановлює як відображати пробіли між словами. Вона приймає наступні значення:
  1. normal - стандартні, автоматичні переноси рядків;
  2. nowrap - не враховує пробіли, у наслідок чого весь текст відображається у вигляді одного рядка;
  3. pre - відображає текст з урахуванням усіх пробілів і переносів рядків, навіть ті, що написані у HTML коді;
  4. pre-line - пробіли не враховуються, але текст автоматично переноситься на новий рядок якщо йому не вистачає місця;
  5. pre-wrap - зберігає всі пробіли и переходи на новий рядок у тексті, автоматично переносить текст, якщо він не вміщається;

Aa

Взаємодія зі шрифтами. Родини шрифтів
Aa
Властивістю font-family можна встановити родину шрифту для тексту.
Існує декілька загальних імен шрифтів:
  1. Serif - шрифти з засічками. До них входять:
    • Times New Roman
    • Georgia
  2. Sans-serif - шрифти без засічок. До них входять:
    • Arial
    • Verdana
  3. Monospace - шрифти у яких всі літери мають однакову ширину. До них входять:
    • Courier New
    • Lucida Console

Спочатку вказується одна або декілька назв сімейств шрифтів через кому, потім загальна назва шрифтів.
Якщо назва сімейства шрифтів з пробілами - необхідно брати її у лапки.
Властивість може приймати декілька шрифтів із одного загального простору імен. Це зроблено для того, щоб якщо у користувача відсутній перший шрифт - браузер вікористав наступний вказаний у властивості.

Стилі тексту
Aa
Загальна стилізація включає в себе декілька властивостей:
Властивість font-style відповідає за написання літер.
Приймає такі значення:
  1. normal - звичайні літери;
  2. italic - нахилені літери;
  3. oblique - нахилені літери (значення дуже схоже до italic, але погано підтримується);

Властивість font-weight відповідає за товщину літер.
Приймає такі значення:
  1. bold - широкі літери;
  2. normal - звичайні літери;
  3. lignter - тонкі літери;

Властивість font-variant відповідає за специфічне написання літер.
Приймає такі значення:
  1. normal - звичайні літери;
  2. small-caps - маленькі літери трансформуються до великих, а великі заміняються такими ж але меншого розміру;
